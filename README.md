sushieval
=========

Sushi evaluation


To generate a total result of duplicate files from `d` file
    $ ./sushieval d

    d
    Total      : 158571
    Unique     : 147936
    Duplicates : 10635, 7%

It will also will generate `d.plot` file

To generate a plot of duplicate files from `d.plot`

   $ ./plot.sh d
